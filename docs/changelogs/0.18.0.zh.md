# v0.18.0 Changelog

这次更新引入了语义搜索功能，并改进了 AI 供应商的配置。

## ✨ Highlights

### Vector Embedding & Semantic Search

- 为文本字段添加了向量嵌入支持，实现了表内语义搜索功能。

### Enhanced AI Provider Configuration

- 改进了 AI 设置页面，包括更好的供应商管理、模型选择、配置表单和成功通知。

## New Features

- **冻结列:** 表格视图现在可以冻结列，以便更好地导航数据。
- **同步 (实验性):** 实现了基础的同步功能，(注意: 同步还在验证中，不会启用。)

## Fixes and Improvements

- 重构了项目结构，将 `locales`, `worker`, `electron`, `lib` 移动到独立的包/应用目录中。
- 修复了字段类型转换失败的问题。
- 修复了网格按钮布局与动态头部高度的问题。
- 修复了设置中的拖动区域行为。
- 更新了英文和中文翻译。
